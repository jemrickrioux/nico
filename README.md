# Nico
## Lis-ça

1. Clone le repo
2. Ouvre le dossier dans ton finder
3. Clique droit sur le fichier main et fait ouvrir (Sinon ça te bloque)
4. Prends connaissance du help menu
5. Va dans ton terminal
6. Roule le script      
        ```$ ./main lire```
7. Réponds aux questions du script
8. C'est tout.
